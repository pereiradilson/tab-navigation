## Prerequisites

- [Node](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/tab-navigation.git

  # Enter folder
  $ cd tab-navigation

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```
